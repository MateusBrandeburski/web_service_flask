Web Service

Este é um projeto desenvolvido para mostar habiliades fullstack com ênfase no backend. Por exemplo, nas caluladoras, temos diversos desafios computacionas com tratamentos de erros ao calcular o Teorema de Pitágoras.

No Web Scraping, é um exemplo simples, mas pode ser expansível para minerar quaisquer dados na web, sem necessidade de escrever todo código novamente, pois existe uma classe para isso.

Já no Crud, também é um exemplo de como persistir dados em um DB na nuvem e imprimi-los na mesma página HTML dinâmicamente e rapidamente.

Foi possível extrair essa velociade em um Web Service que está on-line em uma nuvem Free, que é limitada em velociade. Se fosse feito o deploy desta aplicação em um nuvem paga, seria ainda mais rápida.

Foi usado Flask para o back-end e BootStrap no front-end.

Na rota Web Scraping foi usado Beautifulsoup4 para minerar os dados e na rota SQLAchemy foi usado ele mesmo como ORM.
