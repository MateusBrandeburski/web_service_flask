Web Service

Este é um projeto que tem objetivo de mostrar conhecimentos back-end e fullstack. Por isso, o foco estão nas funcionalidades e no código por baixo da aplicação. Espero que tenha uma boa experiência >.<

Informações Técnicas

Foi usado Flask para o back-end e Bootstrap no front-end.

Na rota Web Scraping foi usado Beautifulsoup4 para minerar os dados e na rota SQLAchemy foi usado ele mesmo como ORM.
